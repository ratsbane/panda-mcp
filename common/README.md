# robot-mcp-common

Shared utilities for `robot-mcp-*` packages. Provides safety validation, computer vision, calibration, scene interpretation, and manipulation planning.

This is a support package used by [robot-mcp-franka](https://pypi.org/project/robot-mcp-franka/) and [robot-mcp-camera](https://pypi.org/project/robot-mcp-camera/). You typically don't install this directly.
